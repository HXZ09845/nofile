# Security Policy

## Supported Versions

NoFile is in an early open-source state. Security fixes target the latest code on `main`.

## Reporting A Vulnerability

Please do not open a public GitHub issue for vulnerabilities.

Use GitHub's private vulnerability reporting feature if it is available on the repository, or contact the maintainer through GitHub.

Helpful reports include:

- Affected version or commit
- Steps to reproduce
- Expected impact
- Any suggested mitigation

## Security Notes

NoFile watches local folders and reads clipboard images when clipboard capture is enabled. Avoid running modified builds from untrusted sources.
