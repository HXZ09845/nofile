# Roadmap

NoFile is early, useful, and still rough. This roadmap keeps the next improvements visible without pretending everything is finished.

## Near Term

- Add real screenshots or a short demo GIF to the README.
- Create a tagged GitHub release with a packaged macOS build.
- Improve first-run permission guidance for macOS.
- Persist user settings and virtual folders more predictably.
- Add basic automated tests for file type detection and watcher behavior.
- Reduce noisy debug logging in production builds.

## Product Polish

- Let users choose multiple watched folders from the UI.
- Add clearer empty states and onboarding copy inside the app.
- Improve keyboard navigation and accessibility labels.
- Add safer handling for missing, moved, or deleted files.
- Add an option to pause clipboard capture.

## Release Quality

- Add signed and notarized macOS builds.
- Add release notes and binary checksums for downloads.
- Add CI jobs for type checking and packaging smoke tests.
- Document privacy behavior for watched folders and clipboard access.

## Ideas

- Rules for auto-grouping screenshots and downloads.
- Search and filtering for recent files.
- Optional pinned items.
- Cross-device or cloud-backed shelves.
